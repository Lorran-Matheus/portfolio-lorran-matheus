Principais tecnologias utilizadas

X CSS (Cascading Style Sheets) é uma linguagem de estilo usada para criar a aparência visual e o layout de documentos web, como páginas HTML, separando o conteúdo da sua apresentação.
X Figma é uma plataforma de design baseada na web, colaborativa e gratuita, utilizada por designers, gestores de produto e desenvolvedores para criar e prototipar designs de interfaces digitais.
Bootstrap é um framework de código aberto popular e gratuito, usado por desenvolvedores Front-end e Full Stack para criar designs de sites e aplicativos responsivos e com boa experiência do usuário.
SASS (Syntactically Awesome Style Sheets) é um pré-processador CSS que adiciona funcionalidades avançadas ao CSS puro, como variáveis, aninhamento e mixins, para tornar o código mais organizado, reutilizável e fácil de manter.
O Git é um sistema de controlo de versão distribuído (DVCS), de código aberto, que permite aos desenvolvedores acompanhar e gerir as alterações nos arquivos de um projeto ao longo do tempo, de forma local e colaborativa.
O GitHub é uma plataforma baseada na nuvem para hospedar, compartilhar e gerenciar código-fonte e projetos de software, funcionando como um hub de colaboração para desenvolvedores.
HTML é a sigla para HyperText Markup Language (Linguagem de Marcação de Hipertexto), a linguagem padrão para criar e estruturar páginas web e o seu conteúdo, como textos, imagens e links.
JS (JavaScript) é uma linguagem de programação interpretada, de alto nível e baseada em objetos, usada principalmente para criar páginas web dinâmicas e interativas.
React é uma biblioteca JavaScript de código aberto, mantida pelo Facebook, utilizada para construir interfaces de usuário (UIs) de forma eficiente e reutilizável através de componentes, sendo especialmente útil para criar aplicações web complexas e interativas, bem como aplicativos móveis com React Native.

  • Navegação interna com menus e redirecionamentos.
  • Acessibilidade e responsividade.

Sobre mim

Meu nome é Lorran Matheus, sou estudante de Análise e Desenvolvimento de Sistemas pela UNISUAM e tenho grande interesse em desenvolvimento. Sou fascinado por compreender como as coisas funcionam e por transformar ideias em soluções práticas.

Atualmente, foco meus estudos em front-end, explorando conceitos como Sass/SCSS, manipulação do DOM, consumo de APIs, versionamento de código e acessibilidade. Recentemente, iniciei meus estudos em React e busco oportunidades para aplicar esse conhecimento em projetos reais.

Meu objetivo é me tornar um desenvolvedor full-stack, mas atualmente meu foco está em aprender, evoluir e contribuir para projetos que façam a diferença. No tempo livre, gosto de explorar novas ideias e transformar pequenos desafios em aprendizado.

  • Manipulação do DOM
  • Criação e uso de event listeners
  • Controle de formulários
  • Uso de localStorage para persistência de dados
  • Uso de fetch() para carregamento dinâmico de componentes e páginas
  • Implementação de IntersectionObserver para detectar elementos visíveis na tela
  • Carrossel/slideshow com classes dinâmicas
  • Tratamento de erros e promises em requisições assíncronas
  • Estruturação modular com export e import entre arquivos JS
  • Uso de condições baseadas em window.innerWidth e matchMedia para adaptar comportamento (mobile vs desktop)

Digital Innovation One - DIO. 07/2025 - 08/2025

Com Sass (SCSS), aprendi a organizar e escalar projetos de estilização através de partials, mixins, variáveis e funções. Isso permitiu estruturar o código de forma modular e reutilizável, facilitando a manutenção e boas práticas. Trabalhei também com media queries organizadas, hierarquia de estilos e metodologias como BEM e ITCSS, otimizando o fluxo de desenvolvimento.

  • Uso de variáveis para cores, tamanhos e espaçamentos
  • Estruturação em partials para organização modular do código
  • Uso de mixins para reutilizar blocos de código
  • Aninhamento de seletores para simplificação da sintaxe
  • Criação de mapas e loops para gerar classes utilitárias dinamicamente
  • Organização de media queries globais e específicas por componente
  • Integração do Sass com compilação automática (watch) para CSS otimizado

Digital Innovation One - DIO. 03/2025 - 07/2025

No curso de CSS, aprimorei minha capacidade de criar interfaces modernas e responsivas. Aprofundei meus conhecimentos em layout utilizando Flexbox e Grid, além de aplicar pseudo-classes, animações e boas práticas de organização e reutilização de estilos. Aprendi também a trabalhar com design tokens, variáveis globais e compatibilidade entre navegadores.

  • Domínio de seletores avançados e pseudo-classes (:not, :has, :where)
  • Uso de Flexbox e Grid Layout para estruturar layouts responsivos
  • Criação de componentes reutilizáveis com classes utilitárias (m-1, p-2, etc)
  • Aplicação de media queries específicas e globais para diferentes breakpoints
  • Uso de transições e animações
  • Estilização de formulários
  • Organização de CSS com metodologias como BEM e ITCSS
  • Criação de carrosséis e modais estilizados
  • Controle de elementos “vazando” do container com overflow
